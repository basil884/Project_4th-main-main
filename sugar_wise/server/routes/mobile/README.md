# Mobile Endpoints Namespace

All mobile app endpoints are grouped under:

- `/api/mobile`

Main sub-routes:

- `/api/mobile/auth` -> register, login, logout
- `/api/mobile/doctors`
- `/api/mobile/patients`
- `/api/mobile/products`
- `/api/mobile/cart`
- `/api/mobile/messages`
- `/api/mobile/notifications`
- `/api/mobile/bookings`
- `/api/mobile/monitoring`
- `/api/mobile/lab-tests`

Example:

- `POST /api/mobile/auth/login`
- `GET /api/mobile/products`
- `GET /api/mobile/messages/chats?userId=...`
